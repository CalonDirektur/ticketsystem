# Helpdesk BFI Syariah

Aplikasi web helpdesk untuk karyawan BFI Syariah untuk pengajuan support produk untuk calon konsumen BFI Syariah dari cabang ke tim head office.

# Teknologi

* PHP
* HTML, CSS, Bootstrap CSS framework
* MySQL Database
